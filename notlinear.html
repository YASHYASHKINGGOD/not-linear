Non-linear chat for research

Alex Chen
14 min read
2 branches

A canvas for non-linear thinking. The premise: most tools force a single thread, but real thinking forks constantly — a sentence sparks a tangent, the tangent has its own life, and you want to chase it without losing the spine.

This document is a chat. You can highlight any sentence to start a side-branch with AI, or just keep talking to it from the composer below. Branches appear as floating panels on the right — collapse them, branch from them again, or promote one as your main context.

The problem

Linear chat conflates two distinct activities: composing a coherent argument and exploring the space around it. When you ask a follow-up, the original thread is poisoned — context shifts, and you can't easily go back to where you were.

The workaround everyone discovers: open a new tab, paste the relevant excerpt, ask the tangent there. Then forget which tab is which. Spatial memory does the heavy lifting only as long as your window manager respects it.

✦
The move. Make branching a one-gesture action — highlight, hit Branch, the side-panel appears. The doc stays where you left it. The branch can branch further, or be promoted to take over the main thread.

Core flows

1. Select → Branch

Highlight any text. A floating popover offers Branch (primary), comment, format. Branch opens a panel on the right with the selected phrase pinned at the top. The new chat is seeded with that quote as context — you ask, AI answers, and the source is always one glance away.

  • Borrowed muscle memory. Same gesture as Notion's comment, just re-aimed.
  • Source becomes context. The branch always knows what it came from.
  • Two clicks max. If it takes more, the gesture has failed.

2. Branch the branch

Inside any branch panel, you can highlight an AI reply and branch again. The new sub-branch stacks on top of the parent. Depth is shown in the header — ↳ 2, ↳ 3 — so you always know where you are.

Or: hit Make this the main thread from a branch's menu, and its conversation replaces the doc as the working surface. The old doc moves to a tab. This is how a tangent that turned out to matter becomes the new spine.

3. The composer is always there

Below this doc is a chat composer. Type into it to keep talking to AI without picking a sentence — useful when you want to add to the document, or generate the next section, or just think out loud. Replies appear inline below this paragraph.

Anti-patterns

  • No tree icons. The doc is the spine. Branches are subordinate.
  • No notifications, badges, streaks. This is a thinking tool. It should never demand attention.
  • No comparison panes inside a branch. If you want comparison, branch as siblings.
